### git本地分支提交顺序
```
git add .                    添加
git commit -m "test"         提交
git checkout master          切换到主分支
git pull                     在主分支同步代码
git merge yourName           将本地自己分支合并到主分支
git push                     将主分支推送到远端 
git checkout yourName        切换到自己分支
git merge master             将主分支合并到自己分支 
```
