### 目录
```
asssets=> front 门户图片
assets=>  img 是后台管理系统
components 公共组件
directive  全局指令
filter     全局过滤器
iconfont   图标
LayoutFront 门户外壳
Layout      后台管理系统目录
pages      页面文件
plugin     全局插件
router => notLogin 门户和不需要登录的路由
router => loginRouter 是后台管理系统路由
store     vuex
style     全局样式
utils     全局工具函数
interceptors  axios配置
permission 路由拦截器
theme    elementui 主题色
```

### 几个路由
```
/index 门户首页路由
/home  管理系统首页路由
/login 登录页  
```
